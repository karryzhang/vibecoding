Company Review Model v2

新版公司评估模型
加入财报、股价、买点

这不是只讲框架的分析模板了，而是一个直接服务买卖决策的版本。以后每次公司评估，都必须同时看公司质量、最新财报、最新股价、估值区间和明确买点，不再停留在“好公司”这种太宽泛的结论上。

框架判断 + 最新事实财报必须更新股价必须更新给出明确买入区间

模型目标

目标不是写漂亮研报，而是给出 可执行 的结论。

新版模型要求每次输出必须回答 4 个问题：公司值不值得长期跟踪？当前财报是否支持逻辑？当前股价是不是好的赔率？如果要买，具体买在哪几个区间？

最终输出必须落到：买 / 观察 / 持有 / 回避，以及明确价格。

不允许只说“公司不错”，不允许只说“长期看好”，也不允许不给价格区间。看对公司但买贵了，一样可能赚不到钱。

四层结构

Layer 1

公司质量层

  • 生意本质
  • 护城河
  • 创始人 / 管理层 / 治理
  • 自由现金流与资本配置
Layer 2

最新事实层

  • 最近一期财报
  • 营收、利润、毛利率、现金流
  • 最新股价、52周区间、估值
  • 最新催化剂 / 风险
Layer 3

估值与买点层

  • 当前价是否合理
  • 合理价 / 偏贵价 / 便宜价
  • 第一买点 / 第二买点
  • 理想重仓区间
Layer 4

操作层

  • 买 / 持有 / 观察 / 回避
  • 建议仓位
  • 加仓触发条件
  • 证伪条件

固定输出字段

模块必须输出目的
结论先行买 / 观察 / 持有 / 回避先给方向，不藏在正文里
一句话生意本质把公司用一句话讲清楚防止自己都没看懂
最新财报摘要营收、利润、毛利率、现金流、核心亮点/问题避免脱离最新事实空谈
最新市场价格当前股价、52周区间、当前估值防止只看公司不看价格
买点建议观察区间、第一买点、第二买点、重仓区把结论变成可执行动作
证伪条件什么情况下判断失效防止信仰化持有

买点建议怎么给

当前价格：XX 观察区间：XX - XX 第一买点：XX 第二买点：XX 理想重仓区：XX - XX 高于 XX：不追 跌破 XX 但基本面未坏：重新评估，不机械接飞刀
买点建议必须基于两个东西同时成立：一是公司质量够好，二是当前价格给到足够赔率。少一个都不行。

同样一家公司，框架判断可以长期稳定，但买点会随着财报、股价和情绪波动不断更新。所以买点不是静态结论，而是动态部分。

评分体系也升级

公司质量分40%
最新财报分25%
估值赔率分25%
催化剂 / 风险分10%

以后不再只给“好公司 / 差公司”这种结论，而是会形成两个分数：公司质量分 和 当前买点分。

这样可以避免一个常见错误：公司明明很好，但当前价格并不值得买，却因为“长期看好”而误买高位。

新版模型的纪律

纪律 1

  • 不引用过旧财报做结论
  • 不使用静态旧价格判断买点
  • 重大事件后要更新

纪律 2

  • 好公司不等于好买点
  • 便宜不等于值得买
  • 估值和质量必须分开判断

纪律 3

  • 必须给明确价位
  • 必须给证伪条件
  • 必须说明什么情况下不买
