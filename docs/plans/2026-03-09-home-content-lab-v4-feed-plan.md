# Home Content Lab V4 Implementation Plan

## 目标
将 `home-content-lab.html` 从混排实验版重构为统一单列信息流版。

## 实施步骤
1. 保留顶部轻壳与底部导航
2. 压缩资产区与快捷操作区的视觉权重
3. 删除双列卡、横滑卡、强调大卡结构
4. 重建统一 feed item CSS 体系
5. 写入 5-6 条示例内容，按时间倒序展示
6. 所有 UI 文案补全中英文映射
7. 自测亮暗主题与移动端节奏
8. 提交并 push 给用户验证

## 验收点
- 页面主视觉是信息流而不是卡片陈列
- 所有内容条目结构统一
- 时间信息清晰可扫读
- 最新内容位于最上方
- CTA 轻但明确
